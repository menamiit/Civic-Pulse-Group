Submission Instructions

Please review these points before submitting your grievance:

  1. Use a clear, short title that identifies the issue quickly.
  2. Select the most relevant category to speed up routing to the right department.
  3. Describe the issue with specifics: what, when, and how severe it is.
  4. Add location details (street, landmark, area) for faster ground action.
  5. Use Current Location if possible for better geo-accuracy.
  6. Upload a photo when available. Clear photos improve resolution time.
  7. Avoid duplicate complaints for the same issue in the same location.
  8. Do not include sensitive personal data in complaint text.
Tip: Complaints with accurate category + location + photo are usually processed faster.
